ical2rss

description

ical2rss is a PHP application that converts an iCalendar file to a RSS stream, showing the 10 upcoming events.

demo

You can use it with an URL like this:
http://shoki.oderland.com/~ixdcthse/ical2rss/ical2rss.php?url=http://path.to/your/icalendar/file.ics
